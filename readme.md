## **Simple Http Server**

A very basic **http server** built with **Node.js** and **MySQL** database.
#### Feature :
  - Basic CRUD operations

#### Installation : 
```sh
npm install
npm start
```

**Request Handle :**
```sh
   http://localhost:3000/
  ```
**Search :**
```sh
  http://localhost:3000/get?id=<int>
   ```
**Delete :**
 ```sh
 http://localhost:3000/delete?id=<int>
  ```
 
**Update :** 
 ```sh
  http://localhost:3000/update?name=<string>&id=<int>
   ```
